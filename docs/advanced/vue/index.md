学习 Vue.js 源码笔记，内容包括：

- 响应式原理

笔记中带有部分个人的理解，并且重点的代码已高亮显示。

Vue.js 2.0 源码的学习是参考黄轶老师的源码解析，他对 Vue.js 源码的解析很到位，是我看到的解析文章中水平较高的一档，对于个人的学习很有参考价值。并且他解析源码的方式跟我个人阅读源码的习惯很相像，知识讲解有进有退，学起来很舒服，这也是我推荐重点学习的资料。
